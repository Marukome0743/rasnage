# rasnage

New Raspi Signage System

## Usage

To clone and run this application, you'll need [Git](https://git-scm.com) and [Bun](https://bun.sh/) installed on your computer. From your command line:

```bash
# Clone this repository
$ git clone https://github.com/OpenUp-LabTakizawa/rasnage

# Install dependencies with bun
$ cd rasnage && bun i

# Develop the app
$ bun --bun run dev

# Build the app
$ bun --bun run build

# Start the app
$ bun --bun run start

# Format the app
$ bun --bun run lint
```
